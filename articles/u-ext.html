Unicode BCP 47 U Extension

The locale extension is an Unicode extension for identifying Unicode locale-based variations using language tags. The extension starts with -u followed by a sequence of name/value pairs.

For example, the ID th-u-nu-thai-ca-buddhist specifies that the Buddhist calendar and the Thai numbering system is used.

The name/value pairs are defined in Unicode TR 35. Some common extension names are:

Name Description Values
ca Calendar algorithm calendar.xml
cf Currency format style currency.xml
ms Measurement system measure.xml
nu Numbering system number.xml
